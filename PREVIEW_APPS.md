## üì± ANDROID PREVIEW

### Option 1: Android Studio Emulator (Recommended)

1. **Open Android Studio:**
   ```bash
   npm run cap:android
   ```

2. **Create/Start Emulator:**
   - Click: **Device Manager** (phone icon)
   - Click: **Create Device**
   - Select: **Phone** ‚Üí **Pixel 5** (or any)
   - Click: **Next** ‚Üí **Finish**

3. **Run App:**
   - Click: **Run** button (green play icon)
   - Or: **Run** ‚Üí **Run 'app'**
   - App will launch in emulator!

### Option 2: Physical Android Device

1. **Enable Developer Mode:**
   - Settings ‚Üí About Phone
   - Tap "Build Number" 7 times
   - Enable "USB Debugging"

2. **Connect Phone:**
   - Plug in via USB
   - Accept debugging prompt on phone

3. **Run:**
   ```bash
   npm run cap:android
   ```
   - Click **Run** in Android Studio
   - Select your device
   - App installs and launches!

## üçé iOS PREVIEW (Mac Only)

### Option 1: iOS Simulator

1. **Open Xcode:**
   ```bash
   npm run cap:ios
   ```

2. **Select Simulator:**
   - Top bar: Select device (iPhone 14, etc.)
   - Or: **Product** ‚Üí **Destination** ‚Üí Choose simulator

3. **Run:**
   - Click: **Run** button (play icon)
   - Or: **Product** ‚Üí **Run**
   - App launches in simulator!

### Option 2: Physical iPhone/iPad

1. **Connect Device:**
   - Plug in via USB
   - Trust computer on device

2. **Configure Signing:**
   - In Xcode: **Signing & Capabilities**
   - Select your Apple ID team
   - Xcode will auto-sign

3. **Run:**
   - Select device in top bar
   - Click **Run**
   - App installs and launches!

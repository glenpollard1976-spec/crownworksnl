## ğŸš€ HOW TO PREVIEW

### Option 1: Web Preview (Easiest)
1. **Build website:**
   ```bash
   npm run build
   ```

2. **Open in browser:**
   - Go to: `http://localhost:3000` (dev mode)
   - Or: `https://crownworksnl.com` (production)

3. **Use browser dev tools:**
   - Press F12
   - Click device toolbar (mobile icon)
   - Select device (iPhone/Android)
   - Preview at different sizes

### Option 2: Android Emulator
1. **Open Android Studio:**
   ```bash
   npm run cap:android
   ```

2. **Run on emulator:**
   - Select device
   - Click Run button
   - App opens in emulator

### Option 3: iOS Simulator (Mac only)
1. **Open Xcode:**
   ```bash
   npm run cap:ios
   ```

2. **Run on simulator:**
   - Select device
   - Click Run button
   - App opens in simulator

### Option 4: Physical Device
1. **Android:**
   - Enable USB debugging
   - Connect phone
   - Run from Android Studio

2. **iOS:**
   - Connect iPhone
   - Trust computer
